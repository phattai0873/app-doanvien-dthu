# Kỹ thuật và Công nghệ (Technical Stack & Skills)

Các kiến thức và công nghệ cốt lõi của dự án được phân bổ như sau:

## 1. Backend Development
- **Node.js (LTS)**: Môi trường thực thi JavaScript chính.
- **Express 5**: Framework web xử lý API.
- **Sequelize ORM**: Quản lý cơ sở dữ liệu PostgreSQL (`pg`).
- **Redis**: Lưu trữ bộ nhớ đệm (Cache) và tối ưu hóa hiệu suất.
- **JWT (JSON Web Token)**: Xác thực người dùng và bảo mật API.
- **Zod**: Thư viện xác thực Schema/Dữ liệu mạnh mẽ.
- **Compression & Rate Limiting**: Tối ưu hóa phản hồi và bảo vệ hệ thống trước tấn công Brute Force.

## 2. Frontend & Web Admin
- **React**: Thư viện UI xây dựng Admin Dashboard.
- **Vite**: Công cụ build hiện đại và máy chủ phát triển nhanh.
- **TailwindCSS**: Framework CSS tiện ích để thiết kế UI nhanh.
- **Vite.js & PostCSS**: Quy trình xử lý Assets.

## 3. Mobile App Development
- **React Native**: Xây dựng ứng dụng di động đa nền tảng (iOS & Android).
- **Expo**: Framework hỗ trợ phát triển và phân phối ứng dụng di động.
- **React Navigation**: Quản lý điều hướng trong ứng dụng.

## 4. Others
- **Git/GitHub**: Quản lý mã nguồn.
- **Docker** (tùy chọn): Triển khai môi trường đồng nhất.
- **PostgreSQL**: Cơ sở dữ liệu quan hệ lưu trữ dữ liệu chính.
