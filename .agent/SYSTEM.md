# Hệ thống Quản lý Đoàn viên - Đại học Tiền Giang (DTHU)

Dự án này là một hệ thống toàn diện để quản lý các hoạt động của Đoàn thanh niên, bao gồm quản lý đoàn viên, hoạt động tình nguyện, tin tức, và tài chính.

## Cấu trúc dự án
- **Backend**: API server xây dựng bằng Node.js và Express.
- **Frontend**: Giao diện web quản trị (Admin Web) sử dụng React và Vite.
- **Mobile**: Ứng dụng dành cho đoàn viên trên nền tảng React Native (Expo).
- **LandingPage**: Trang giới thiệu dự án.

## Mục tiêu hệ thống
Cung cấp công cụ quản lý hiện đại, bảo mật và hiệu quả cho các cán bộ Đoàn và đoàn viên, hỗ trợ thanh toán phí qua mã QR và theo dõi hoạt động tình nguyện.
